Definición general del proyecto de software

  1. Idea General:
    • ​La idea general del este proyecto es la creación de un sistema en el cual se pueda almacenar paginas web para tener una biblioteca de recursos, ya sea multimedia,de artículos  etc. En los que se pueda interesar o apoyarse en un futuro. Mediante módulos y etiquetas que permitan si identificación y clasificación pertinentes.
  2. Objetivos:
    • ​Objetivo Generales:
      • ​Buscar y seleccionar recursos disponibles en la red para incorporarlos a sus propias bibliotecas virtuales.
      • Crear aplicaciones sencillas de software estructurado utilizando un Entorno de Desarrollo Integrado orientadas a la producción de una página web interactiva.
      • proporcionar un espacio personalizable para la organización de material recolectado en la web.
    • Objetivos Específicos:
      • ​Visualizar, crear y buscar información en Internet para almacenar en un biblioteca virtual de recursos
      • Diseñar un Sitio Web para organizar y acceder de una forma cómoda a la información utilizando lenguajes de programación y de presentación (html, css, javascript y php)
      • clasificar las paginas recolectadas para poderlas visualizar de manera mas ordenada.
      • Diseñar una base de datos optima e implementar un sistema de auditoria a la misma.
      • Diseñar una interfaces gráfica amigable e intuitiva.
    • ​Usuarios:
      • ​Este sistema es diseñado para el usuario promedio, que le gusta entrar a Internet desde una PC o portátil, desde un navegador ya sea chrome o opera. 

Especificación de requerimientos:

  1. ​Requerimientos generales:
    • ​Identificación de los distintos procesos necesarios para el almacenamiento de contenido web con su respectivas características y sus posibles aplicaciones.
    • Determinar la cantidad de paso que se tiene que hacer para este proceso de manera manual.
    • Asegurarse de que se dispone de recursos  y medios necesarios para apoyar los procesos y realizar el seguimiento de los mismos.
    • La información y análisis dará lugar a acciones para rectificar y alcanzar los resultados planificados, siempre bajo la máxima de mejora continua.
    • Monitorear y analizar los procesos de forma continua, para lo cual se precisa de un seguimiento, monitoreo y análisis de estos procesos.En ésta etapa hay que identificar la información y resultados que interesa conocer, como registros y el análisis de la mismo.
